# Rishan Portfolio

Premium developer portfolio with dark futuristic AI SaaS design.

## Tech Stack

- **Frontend:** Next.js 14 + React + TypeScript
- **Styling:** CSS Modules + Framer Motion
- **Backend:** Next.js API Routes
- **Email:** Nodemailer

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

## Environment Variables

Create `.env.local`:

```
SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_USER=your-email@gmail.com
SMTP_PASS=your-app-password
ADMIN_EMAIL=your-email@gmail.com
```

## Features

- Dark futuristic AI SaaS design
- Animated fluid wave background
- Rotating word hero animation
- Glassmorphism cards
- Glowing CTA buttons
- Responsive layout
- Contact form with email

## Deploy

Deploy to Vercel - just connect the repo and add env variables.
